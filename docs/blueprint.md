# **App Name**: ArroBuddy

## Core Features:

- Instant Feed Trigger: A 'Feed Now' button to remotely trigger the pet feeder for immediate food dispensation.
- Feeding Schedule Management: Create, edit, and delete daily automated feeding schedules for your pet.
- Real-time Food Level Monitoring: Display current food levels in the feeder using ultrasonic sensor data from the ESP32.
- Feeder Status Dashboard: View essential information such as last fed time and current schedule status in a concise dashboard.
- Secure User Authentication: Login functionality to protect access to the feeder controls.
- Remote Device Communication: Integration with an ESP32 micro-controller to send feeding commands and receive sensor data via Firebase Realtime Database.
- Personalized Feeding Insights Tool: An AI-powered tool that analyzes feeding patterns and suggests tailored tips for optimal pet health and feeding routines.

## Style Guidelines:

- Primary color: A vibrant, reliable blue (#4280E5) that signifies trustworthiness and technology, contrasting with a dark background.
- Background color: A very dark, subtle blue-grey (#15181C) to provide a modern, sleek foundation for the dark theme.
- Accent color: A bright, clear sky-blue (#98E2F6) used for highlighting interactive elements and providing visual separation.
- Headlines and body text: 'Poppins', a geometric sans-serif, for a modern, precise, and approachable feel, fitting the clean aesthetic.
- Monospaced elements: 'Source Code Pro', a monospaced sans-serif, specifically for displaying time, sensor values, and other technical readouts, as indicated in the reference code.
- Use clear, outline-style icons from the Lucide set for readability and a contemporary visual language, consistent with the provided code examples.
- Employ a responsive, card-based layout centered on content, optimizing for clear information hierarchy and intuitive interaction across devices.
- Subtle, deliberate animations for state changes (e.g., loading spinners, low food alerts), providing clear visual feedback without distracting the user.
